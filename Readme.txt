Spanish:

Cantidad a enviar por defecto tienen que poner 500
Cantidad total la cantidad que se deasea enviar por ejemplo: 20000

English:

The default amount to send should be set to 500
The total amount is the desired amount to send, for example: 20000


Pips que necesitas instalar:

pip uninstall amino.fix
pip uninstall BotAmino
pip install tqdm
pip install https://github.com/ThePhoenix78/BotAmino/archive/master.zip
pip istall request
pip install colorama
pip install colored
pip install pystyle

En el main.py estan todos solamente tienes que poner (y)
